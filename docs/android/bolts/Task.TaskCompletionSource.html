Class Task.TaskCompletionSource

Object
  bolts.Task.TaskCompletionSource
Enclosing class:
Task<TResult>
public class Task.TaskCompletionSource
extends Object

Allows safe orchestration of a task's completion, preventing the consumer from prematurely completing the task. Essentially, it represents the producer side of a Task, providing access to the consumer side through the getTask() method while isolating the Task's completion mechanisms from the consumer.
